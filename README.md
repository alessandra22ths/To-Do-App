# To-Do-App
Ce projet est une petite application développée dans un objectif d’apprentissage. Il s’agit d’un programme en console qui permet de gérer une liste de tâches de manière simple, directement depuis le terminal.

L’utilisateur peut ajouter des tâches, définir une priorité, consulter la liste des tâches et marquer certaines d’entre elles comme complétées. Les données sont sauvegardées dans un fichier texte afin de pouvoir retrouver les tâches lorsque le programme est relancé.

Ce projet m’a permis de travailler la logique de programmation, la gestion des données, ainsi que la structure d’un programme interactif. J’ai également pratiqué la lecture et l’écriture de fichiers, ainsi que la gestion des entrées utilisateur.

L’objectif principal était de renforcer mes bases à travers un projet concret, sans chercher à faire quelque chose de complexe, mais fonctionnel et utile pour l’apprentissage.
